# FASTQ Dataset Preparation for Brain Cancer Classification

## 1. Biological Problem
The project addresses the challenge of Early Genomic Detection of Brain Cancer (specifically Glioblastoma). The primary goal is to distinguish between healthy neural tissues and malignant tumors by analyzing DNA sequencing patterns. In brain cancer, specific genetic alterations occur, such as point mutations (SNPs) and structural changes, which can be identified within the raw FASTQ data.

## 2. Classes
The dataset consists of two distinct biological classes for binary classification:

Class 0 (Healthy Samples)
Class 1 (Brain Cancer Samples)

## 3. Why the Dataset is Useful
This dataset serves as a critical foundation for bioinformatics applications such as:

Machine Learning Training: It provides labeled data to train classification models that can automatically detect cancer signatures in genomic reads.
